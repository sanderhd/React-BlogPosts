# React Eindopdracht

Dit is mijn eindopdracht voor React. Het is een eenvoudige blog waar gebruikers berichten kunnen maken en lezen. Het doel van dit project is om de basisprincipes van React te leren, zoals componenten, state en props.

## Functies
- Blogberichten maken en lezen
- Eenvoudig en schoon ontwerp
- Gebouwd met React en moderne JavaScript

## Presentatie
Aan het einde van deze periode zal ik dit project presenteren en uitleggen hoe het werkt en wat ik heb geleerd.

## Toekomstige Plannen
- Berichten bewerken en verwijderen
- Categorieën of tags toevoegen

## Gebruikte Technologieën
- React
- React Router Dom
- JSON

## Installatie
Om de benodigde pakketten te installeren, voer uit:
```sh
npm install
```

## Project Starten
Om de ontwikkelserver te starten, voer uit:
```sh
npm start
```

## Gebruik
1. Open de applicatie in je browser.
2. Bekijk bestaande blogberichten.
3. Klik op de knop "Create a new Post" om een nieuw blogbericht toe te voegen.
4. Vul de titel en inhoud in en dien het formulier in om het bericht toe te voegen.
